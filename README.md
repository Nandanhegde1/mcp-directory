# MCP Directory

A static, SEO-first directory of every public Model Context Protocol (MCP) server on GitHub. Built with Astro, deployed free on Cloudflare Pages, auto-updated weekly via GitHub Actions.

## Quick start

```powershell
cd S:\projects\mcp-directory
npm install
# 1) Generate a fine-grained GitHub PAT with public repo read access:
#    https://github.com/settings/personal-access-tokens/new
$env:GITHUB_TOKEN = "github_pat_..."
npm run scrape          # populates src/data/servers.json
npm run dev             # local preview at http://localhost:4321
npm run build           # outputs static site to ./dist
```

## Project structure

```
src/
  data/servers.json      # generated by scripts/scrape.ts
  layouts/BaseLayout.astro
  pages/                 # routes
  lib/data.ts            # data accessors
  components/
  types.ts
scripts/
  scrape.ts              # GitHub MCP server scraper
  featured.json          # array of slugs to mark as featured
public/
  robots.txt, favicon.svg, og-default.svg
.github/workflows/
  weekly-scrape.yml      # cron: re-scrape every Sunday
```

## Environment variables

Set these in Cloudflare Pages → Settings → Environment variables (and locally in `.env`):

| Var | Purpose | Required? |
| --- | --- | --- |
| `SITE_URL` | Canonical URL (e.g. `https://mcp-directory.com`) | Recommended |
| `PUBLIC_FORMSPREE_ID` | Formspree form ID for `/submit/` | Optional |
| `PUBLIC_STRIPE_FEATURED_LINK` | Stripe Payment Link for `/featured/` | Optional |
| `PUBLIC_ADSENSE_CLIENT` | `ca-pub-XXXXXXXXXX` for AdSense | Optional |
| `PUBLIC_AFFILIATE_ANTHROPIC` | Affiliate URL override (if you have one) | Optional |
| `GITHUB_TOKEN` | Used by scraper script (local + CI) | Required for scraping |

For the GitHub Action, store the token as repo secret `SCRAPER_GITHUB_TOKEN` (or rely on the default `GITHUB_TOKEN`, which has lower rate limits).

## Deploy to Cloudflare Pages

1. Push the repo to GitHub.
2. Cloudflare Pages → Create project → Connect to GitHub → pick this repo.
3. Framework preset: **Astro**. Build command: `npm run build`. Output dir: `dist`.
4. Add the env vars above (at minimum `SITE_URL`).
5. Deploy. Future commits to `main` auto-deploy.

## Monetization

- **Featured listings** (`/featured/`): $29 / 30 days. After payment, add the slug to `scripts/featured.json` and push.
- **Affiliate links**: edit `BaseLayout.astro` / detail page CTA.
- **AdSense**: set `PUBLIC_ADSENSE_CLIENT` and approval will inject the script.

## Maintenance

The weekly cron rebuilds `servers.json` and pushes it. Cloudflare auto-redeploys.

User actions per month (~20 min total):
- Approve any submitted servers (Formspree → notes).
- Add featured slugs after Stripe payments.
- Glance at analytics + revenue.
